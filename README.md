# FILE EXPLORER


- Expandable Sidebar (The Tree View)
- Main Content Area displays the contents of the currently selected folder
- List View: A detailed table (Name, Type, Date Modified).
- Grid View: Icon-based grid layout.
- Colour-Highlighted Hierarchy (Level 1 is dark teal colour, Level 2 is medium teal, Level 3 is light teal, etc.).
- Navigation: Clicking a folder in the tree or main view opens it.
- CRUD Operations
- Create a new File or Folder inside the current directory.
- Rename an item.
- Delete an item.
- Search.
